💀 ATTACKER ZONE — Kali Linux VM Tools

  • Reconnaissance: nmap -sV -A -p- 192.168.56.102
  • Web Scan: nikto -h http://target | gobuster dir -u target -w wordlist
  • SQL Injection: sqlmap -u "http://target/login?id=1" --batch --level=3
  • XSS/CSRF: Burp Suite intercept → inject script payload
  • SSH Brute: hydra -l root -P rockyou.txt ssh://192.168.56.102
  • Port Scan: nmap -sS -T4 --open 192.168.56.0/24
  • DDoS: hping3 --flood -S --rand-source 192.168.56.102
  • ARP Spoof: arpspoof -i eth1 -t target gateway
  • MITM: ettercap -T -q -M arp:remote /target/ /gateway/
  • Phishing: GoPhish → campaign → credential harvest

🛡️ DEFENSE PERIMETER — Ubuntu VM Tools

  • Suricata IDS: /var/log/suricata/eve.json → real-time alerts
  • Snort IPS: /var/log/snort/alert → signature matching
  • Fail2ban: /var/log/auth.log → 5 fails → iptables DROP
  • Cowrie Honeypot: Port 2222 fake SSH → log all commands
  • WAF (ModSecurity): HTTP layer 7 SQLi/XSS/CSRF block
  • pfSense Firewall: Stateful packet filtering + NAT
  • iptables: Auto-block rules from Fail2ban/Suricata
  • AES-256: Encrypt sensitive data at rest
  • TLS 1.3: Encrypt data in transit
  • Forwarder: aegis_forwarder.py → POST to AEGIS API

⚡ AEGIS BRAIN — SOC Engine Components

  • API Server: Express 5 + Node.js 24 + TypeScript
  • Database: PostgreSQL + Drizzle ORM (7 tables)
  • Ingest Auth: X-AEGIS-Key header validation
  • Event Parse: Snort/Suricata/Fail2ban/Cowrie normalizer
  • Severity Score: critical/high/medium/low auto-classify
  • Alert Engine: Critical/High → auto Alert record create
  • Incident Correlate: Same IP + 5min window → Incident
  • SSE Broadcast: Real-time push to all dashboard clients
  • Network Track: network_hosts table → topology data
  • Defense Log: blocked_ips + defense_actions tables
